Ai-agent

本地浏览器里的 Coding Agent 控制台。Cursor 走云端 SDK；OpenAI / DeepSeek 走兼容 API + 本地工具。

环境要求

  • Python 3.10+（cursor-sdk 要求）
  • Cursor API Key（Integrations）

快速开始

git clone <this-repo-url> Ai-agent
cd Ai-agent

conda create -n ai-agent python=3.10 -y
conda activate ai-agent
pip install -r requirements.txt

cp .env.example .env
# 编辑 .env，填入 CURSOR_API_KEY

./run.sh

自检：

python -m backend.check

浏览器：

http://127.0.0.1:8765/       # 本页
http://127.0.0.1:8765/cursor # Cursor

Agent 默认工作区是本仓库（host_project_root: "."）。要改其它目录，编辑 config.yaml。

HTTP API

curl -X POST http://127.0.0.1:8765/api/chat \
  -H 'Content-Type: application/json' \
  -d '{"message":"列出工作区顶层目录"}'

配置说明

字段说明
host_project_rootAgent 工作区根目录，相对本仓库；默认 "."
server.host / server.port服务地址，默认 0.0.0.0:8765
agent.providercursor / openai / deepseek
agent.model默认 composer-2.5
agent.runtimelocal 或 cloud

路由

  • GET / — 本页
  • GET /cursor — Cursor 对话
  • GET /openai — OpenAI Agent
  • GET /deepseek — DeepSeek Agent
  • GET /api/health — 健康检查
  • POST /api/chat / /api/chat/stream — 对话（body 可带 provider）

安全

  • Agent 可读写工作区并执行命令，仅建议本机使用
  • 公网请改绑 127.0.0.1；勿提交 API Key
