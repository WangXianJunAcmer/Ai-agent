Ai-agent

一个可嵌入任意项目的 Cursor Dev Agent sidecar。它可以作为独立说明页打开，也可以通过 widget.js 注入到宿主项目页面。

Checking... 查看健康检查 JSON

快速开始

  1. 在 .env 里填入 CURSOR_API_KEY。
  2. 运行 bash run.sh 启动服务。
  3. 访问当前页面确认服务和网络都正常。
  4. 需要嵌入宿主项目时，在页面模板里引入 /static/widget.js。

接口说明

  • GET /api/health：健康检查
  • POST /api/chat：同步对话
  • POST /api/chat/stream：SSE 流式对话
  • GET /static/widget.js：可嵌入聊天组件

直接嵌入宿主项目

把下面这行插入到宿主项目页面模板的 </body> 前：

<script src="http://10.130.214.58:8765/static/widget.js"
  data-api-base="http://10.130.214.58:8765"></script>

命令行测试

curl -X POST http://127.0.0.1:8765/api/chat \
  -H 'Content-Type: application/json' \
  -d '{"message":"先看看这个项目是干什么的"}'
