Ai-agent

一个可嵌入任意项目的 Cursor Dev Agent sidecar。你可以把它当成一个独立的本地服务，也可以把聊天入口嵌入到宿主项目页面里。

Checking... 查看健康检查 JSON

快速开始

1. 配置密钥 在 .env 中填入 CURSOR_API_KEY。
2. 启动服务 执行 bash run.sh，默认端口是 8765。
3. 访问首页 用当前页面确认网络、模型和 API 都正常。
4. 嵌入项目 在宿主项目模板中引入 /static/widget.js。

接口说明

浏览器能直接打开的是 GET 路由；聊天接口需要用 POST 调用。

  • GET /api/health：健康检查
  • POST /api/chat：同步对话
  • POST /api/chat/stream：SSE 流式对话
  • GET /static/widget.js：可嵌入聊天组件

直接嵌入宿主项目

把下面这行插入到宿主项目页面模板的 </body> 前。宿主页面右下角会出现一个 AI 按钮。

<script src="http://10.130.214.58:8765/static/widget.js"
  data-api-base="http://10.130.214.58:8765"></script>

命令行测试

不嵌页面时，也可以直接用 API 验证 Agent 是否能工作。

curl -X POST http://127.0.0.1:8765/api/chat \
  -H 'Content-Type: application/json' \
  -d '{"message":"先看看这个项目是干什么的"}'
